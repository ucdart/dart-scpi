# DART SCPI
GUI for Agilent E5071A VNA

![GitHub Logo](/pictures/cavity.png)

## Installation
Make sure you have Microsoft .NET Framework 4.52 installed.

To install please download "setup.msi" directly from the project root and double click it, then follow instructions below.

### Step One
Click Next after seeing the Welcome Screen.
<br/><img src="https://github.com/ucdart/dart-scpi/blob/master/pictures/install0.png" width="500">

### Step Two
Please accept the License Agreement.
<br/><img src="https://github.com/ucdart/dart-scpi/blob/master/pictures/install1.png" width="500">
<br/><img src="https://github.com/ucdart/dart-scpi/blob/master/pictures/install1-0.png" width="500">

### Step Three
Please select the installation location.
<br/><img src="https://github.com/ucdart/dart-scpi/blob/master/pictures/install2.png" width="500">

### Step Four
Please confirm the installation by clicking next.
<br/><img src="https://github.com/ucdart/dart-scpi/blob/master/pictures/install3.png" width="500">

### Step Five
Please allow elevated priviledge for the installer.
<br/><img src="https://github.com/ucdart/dart-scpi/blob/master/pictures/install4.png" width="500">

### Step Six
Please wait until this screen shows up and click [Close] to finish installation.
